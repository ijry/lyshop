# 功能介绍

LYShop 以电商交易链路为核心，提供以下业务能力。

## 用户与认证

- 用户登录与会话管理
- 管理员认证与权限隔离
- 多端统一鉴权入口

## 商品与内容

- 商品、分类、SKU、图片管理
- 装修页面配置
- 商品列表、详情、推荐能力

## 订单与履约

- 购物车、下单、订单状态流转
- 地址管理、订单查询
- 库存联动（WMS 插件）

## 营销与增长

- 优惠券、活动、积分等营销能力
- 秒杀与促销页面承载

## 支付与生态

- 微信支付、支付宝等支付插件
- 本地/云存储扩展
- 短信、OAuth、AI 图片等扩展能力

## 即时通讯

- IM 会话与消息基础能力
- 商家与用户消息场景支持
