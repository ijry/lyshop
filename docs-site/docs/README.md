# docs-site 说明

`docs-site` 是 LYShop 的官网文档站，基于 VitePress 构建。

## 启动命令

```bash
npm install
npm run docs:dev
```

## 构建命令

```bash
npm run docs:build
npm run docs:preview
```

## 目录说明

- `docs/.vitepress/config.mts`：站点配置
- `docs/index.md`：首页
- `docs/guide`：功能介绍
- `docs/deploy`：部署文档
- `docs/api`：接口文档
- `docs/dev`：二次开发
