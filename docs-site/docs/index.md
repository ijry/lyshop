## 最近更新（2026-05-24）

- H5 商品列表升级为 `u-tabs + up-waterfall`，修复 tab 样式异常与瀑布流展示一致性。
- H5 首页轮播修复空白问题：装修数据兼容数组/字符串格式，并改用本地静态轮播资源。
- H5 收货地址支持新增提交，演示模式与生产模式都可获得可见成功反馈。
- H5 订单页演示数据支持按状态过滤，顶部 tab 点击切换立即生效。
- H5 客服会话支持直接输入并发送，WebSocket 不可用时自动降级本地回复。
- PC 端客服入口统一改为页面内弹窗，不再打开新页面。
