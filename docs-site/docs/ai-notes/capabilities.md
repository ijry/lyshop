# 能力总览

本页基于 `guide/features.md` 与仓库 `README.md` 的现有描述整理 AI 生图能力。

## 核心能力

- 多模型聚合：通义万象 / 文心一格 / 腾讯混元 / OpenAI DALL-E。
- 生成目标：封面图、轮播图、详情图等商品素材。
- 参考图能力：按模型能力控制是否支持参考图输入。

## 业务集成位置

- AI 生图能力已内嵌到商品编辑页。
- 后台不再保留独立 AI 生图菜单入口。

## 典型使用链路

1. 在后台配置可用模型与密钥。  
2. 在商品编辑流程中选择用途并生成素材。  
3. 将生成结果应用到商品封面、轮播图或详情内容。  

## 边界与注意事项

- 本页描述的是“当前文档中已声明能力”，不代表未写入文档的隐含能力。
- 具体字段与接口参数以接口文档为准。
